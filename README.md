# BeeVenture


1. Virtual Environment Create
```
python -m venv gameDev
```

2. Virtual Environment Activate
```
source gameDev/bin/activate
```

3. Install requirements
```
pip install pygame mpu6050-raspberrypi RPi.GPIO
```

4. Run app
```
python app.py
```
